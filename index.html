About this page

This project is meant to be a recreation of the arcade game Flappy Bird. It displays the familiar background of the game, along with one of the original flappy bird images, both of which files are provided. The objective of the game is for the bird to go through the gap of the obstacles, without touching it, by continuously pressing the spacebar.

Installation

The first step to installing this game is to install pygame on your command prompt (if playing the game using it).

Download the python file, along with image files to go along with it. Keep these files together in the same folder.

Usage

After installation, one way this project can be used is by command prompt. Open command prompt and navigate to the directory where the python file is held. Once the cmd is on the directory, enter the name of the project with the file extension (i.e. flappybird.py). The game window should be opened for you to play.

Another way is through a Python shell. Simply open the project through a python IDLE, then press run.
